# Visualization Techniques Summary

| Technique        | Best Used For                    | Key Characteristic                             | How it Differs from Similar Charts                                                                                                  |
|:-----------------|:---------------------------------|:-----------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| **Bar Graph**    | Comparing categories             | Discrete rectangular bars                      | Has gaps between bars (unlike Histograms).                                                                                          |
| **Histogram**    | Data distribution                | Bins of continuous data ranges                 | Bars touch to show continuous ranges (unlike Bar Graphs).                                                                           |
| **Line Chart**   | Trends over time                 | Points connected by lines                      | Cleaner for multiple overlapping trends than Area Graphs.                                                                           |
| **Area Graph**   | Volume over time                 | Line chart with filled space below             | Emphasizes magnitude/volume rather than just the trend line.                                                                        |
| **Pie Chart**    | Static proportions               | Slices of a circular whole                     | Only shows a single snapshot in time (unlike Stacked Area).                                                                         |
| **Stacked Area** | Proportions over time            | Layered cumulative areas                       | Combines a trend line's timeline with a pie chart's parts-to-whole view.                                                            |
| **Scatter Plot** | Finding correlations             | Individual X-Y coordinate dots                 | Treats both axes as independent variables (unlike Line Charts).                                                                     |
| **Bubble Chart** | 3-Variable correlations          | Scatter plot with varying point sizes          | Adds a third dimension (size) to a standard Scatter Plot to represent an additional variable (like stress level).                   |
| **Stem Chart**   | Discrete specific values         | Vertical line topped with a marker             | Thinner than a Bar Graph, highlighting the exact data point over volume.                                                            |
| **Step Chart**   | Discrete changes over time       | Points connected by horizontal/vertical steps  | Shows exact periods where a value remained constant before abruptly changing (unlike a slanted Line Chart).                         |
| **Box Plot**     | Statistical distribution summary | Box with whiskers showing quartiles & outliers | Focuses strictly on statistical summary ranges (median, quartiles) rather than showing the full frequency shape (like a Histogram). |
| **Violin Plot**  | Full density distribution        | Mirrored smooth density curves                 | Combines the statistical summary of a Box Plot with the full, curved distribution shape of a density plot/Histogram.                |
