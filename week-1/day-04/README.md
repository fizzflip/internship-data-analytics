## Summary

Day 4 of my Data Analytics internship introduced the fundamentals of data visualization. The primary focus was utilizing
the `matplotlib` library to transform raw numerical data and data structures into intuitive visual representations. I
explored various chart types, including bar graphs, line charts, and pie charts, learning how to customize them to
communicate data insights effectively.

## Key Learnings

### Data Visualization with Matplotlib

* **Library Integration:** Learned to import and utilize the `matplotlib.pyplot` module for creating standard data
  visualizations in Python.
* **Chart Customization:** Practiced enhancing charts with descriptive metadata, including titles (`plt.title()`), axis
  labels (`plt.xlabel()`, `plt.ylabel()`), and custom color schemes.
* **Data Mapping:** Explored different methods of feeding data into plotting functions, utilizing both standard Python
  lists and dictionary views (`keys()` and `values()`).

### Core Chart Types

* **Bar Graphs:** Utilized `plt.bar()` to compare categorical data quantities, passing arrays for categories and their
  corresponding numerical values.
* **Line Charts:** Implemented `plt.plot()` to visualize trends and progressions over sequential data points, such as
  time series (days or months).
* **Pie Charts:** Created proportional representations using `plt.pie()`, learning to map labels to specific slices and
  automatically calculate and display percentage values using the `autopct` formatting parameter.
