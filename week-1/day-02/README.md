## Summary

Day 2 of my Data Analytics internship focused heavily on mastering Python control flow, looping structures, and handling user input. Building upon the previous day's foundation, I applied these fundamental programming concepts to create basic interactive scripts, concluding with building an interactive search tool for querying live API data.

## Key Learnings

### Control Flow and Looping

* **Conditional Logic:** Implemented `if-elif-else` statements to handle various logical paths and game outcomes based on user variables.
* **Iteration:** Utilized `for` loops to iterate a fixed number of times using `range()` and to traverse elements within a list.
* **User Input Processing:** Practiced prompting for user input, performing type casting to appropriate data types like integers, and utilizing this input to steer programmatic logic.
* **State Management:** Integrated boolean flags to efficiently track conditions across iterations or different logic blocks.

## Practical Application

### Interactive Scripts and API Search Tool

The practical tasks involved building interactive command-line programs.

* **Game Logic Implementation:** Created a number-guessing game and a lottery checking system. This required integrating control flow with user inputs to dynamically determine outcomes (e.g., win/loss states, too high/too low feedback).
* **Interactive Data Searching:** Combined external API requests with user input loops to create a search tool.
* **Data Processing:** Iterated through structured JSON cryptocurrency data to find and display records matching the user's search queries.
