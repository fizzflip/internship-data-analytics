## Summary

Today marked the beginning of my Data Analytics internship. The primary focus of Day 1 was establishing a strong
foundation in Python data structures, moving from basic dictionary implementations to navigating complex nested
structures. The day concluded with a practical assignment where I applied these concepts to fetch and process data from
a live cryptocurrency API.

## Key Learnings

### Python Dictionaries & Data Structures

* **Foundations:** Reviewed standard dictionary creation and key-value pair assignments.
* **String Formatting:** Used f-strings to format output dynamically.
* **Navigating Nested Data Structures:** Learned how to systematically unpack multi-layered objects containing lists of
  dictionaries and nested keys.
* **Mixed Data Structures:** Worked with data spanning multiple levels of complexity, organizing variables as
  combinations of arrays and dictionaries to model real-world scenarios.

## Practical Application

### Live Cryptocurrency API Parsing

The milestone task for the day was connecting to a live API to extract cryptocurrency data.

* **Data Ingestion:** Used the `requests` library to fetch a real-time JSON payload from a public cryptocurrency
  endpoint.
* **Nested Parsing:** The API response returned a nested structure containing cryptocurrency market information. I
  successfully navigated the response to extract specific values.
* **Data Extraction:** Extracted data points such as the specific USD price of a coin.
