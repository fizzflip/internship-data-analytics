# Week 1: Python Fundamentals and Data Visualization

## Summary

The first week of my Data Analytics internship focused on establishing a robust foundation in Python programming and introductory data visualization. The week began with mastering core Python concepts, ranging from complex data structures and control flow to building interactive command-line applications. Midweek, the focus shifted toward real-world data acquisition by making HTTP requests to live APIs and parsing nested JSON responses. The week concluded by transitioning from data processing to data visualization, utilizing the `matplotlib` library to transform raw data into insightful, customized charts and graphs.

## Daily Breakdown

### Day 1: Dictionaries & Nested Data Structures
- Explored fundamental Python dictionaries, string formatting, and nested data structures.
- **Practical:** Connected to a live cryptocurrency API and successfully navigated a complex JSON payload to extract specific market data.

### Day 2: Control Flow & Looping
- Mastered conditional logic (`if-elif-else`) and iterative loops (`for`).
- **Practical:** Built interactive scripts, including a number-guessing game and a dynamic API search tool that queried live data based on user input.

### Day 3: Lists & CLI Programs
- Practiced list manipulation, priority sorting using lambdas, and continuous execution with `while True` loops.
- **Practical:** Developed a menu-driven To-Do List manager and created search tools to navigate JSON arrays from the ISRO and Mutual Fund APIs.

### Day 4: Introduction to Data Visualization
- Integrated the `matplotlib.pyplot` library to communicate data insights visually.
- **Practical:** Created and customized foundational chart types, including bar graphs for categories, line charts for trends, and pie charts for proportional data.

### Day 5: Advanced Visualization & Analysis
- Progressed to visualizing data distributions and correlations.
- **Practical:** Built histograms to analyze frequency distributions, plotted multiline charts for direct comparisons, and utilized scatter plots to identify relationships across multiple variables.

## Skills Acquired

* **Python Data Structures:** Advanced manipulation of lists, tuples, and nested dictionaries.
* **Control Flow & Logic:** Conditional branching, iteration, state management, and lambda functions.
* **API Integration:** Fetching, parsing, and searching nested JSON data using the `requests` library.
* **Data Visualization:** Creating and styling Bar, Line, Pie, Scatter, and Histogram charts using `matplotlib`.
