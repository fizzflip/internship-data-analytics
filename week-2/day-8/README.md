## Summary

Day 8 of the Data Analytics internship marked the introduction to the powerful `pandas` library, focusing on
foundational data structures and data manipulation techniques. The tasks centered around creating, exploring, and
modifying Pandas Series and DataFrames to analyze structured data sets effectively.

## Key Learnings

### Pandas Fundamentals

* **Core Data Structures:** Introduced to the two primary Pandas objects: 1-dimensional `Series` and 2-dimensional
  tabular `DataFrame` structures.
* **Data Initialization:** Practiced constructing DataFrames from various Python data types, including lists,
  dictionaries, and nested 2D lists, while explicitly defining custom row indices and column headers.
* **Data Exploration Methods:** Learned essential functions to inspect and understand dataset properties, such as
  `.info()` for schema details, `.describe()` for statistical summaries, `.shape` for dimensions, and `.head()`/
  `.tail()` for quick previews.

## Practical Application

### Data Manipulation Exercises

* **Cricketer Stats DataFrame:** Created a DataFrame from a dictionary mapping to organize player statistics,
  differentiating between batsmen and bowlers.
* **Student Grades Analysis:** Built a comprehensive tabular DataFrame representing student scores across multiple
  subjects to practice utilizing built-in exploration functions (info, describe, shape).
* **Salary Projections Operations:** Constructed a job salary DataFrame containing historical pay data. Performed
  column-wise mathematical operations to project future salaries (adding a 12% increase for a future year) and aggregate
  total cumulative earnings across multiple columns.
