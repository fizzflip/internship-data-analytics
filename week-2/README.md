# Week 2: Advanced Visualization and Introduction to Pandas

## Summary

The second week of the Data Analytics internship transitioned from foundational Python to advanced data visualization
and the introduction of the powerful `pandas` library. The first half of the week was dedicated to exploring specialized
charting techniques using `matplotlib`, such as plotting statistical distributions and multivariable correlations. By
the end of the week, the focus shifted towards robust data manipulation, introducing core `pandas` data structures (
Series and DataFrames) to organize, explore, and analyze tabular data sets.

## Daily Breakdown

### Day 6: Advanced Matplotlib Charts

- Transitioned beyond basic line/bar graphs to visualize complex datasets.
- **Practical:** Created overlapping Area Graphs for stock trends, Multi-scatter plots for behavioral correlations,
  Stacked Plots for cumulative sales, and Stem Charts for time-series event tracking.

### Day 7: Statistical Visualizations

- Focused on charts designed to represent statistical distributions, ranges, and variations within data groups.
- **Practical:** Developed Box Plots to visualize salary ranges across departments, Violin Plots to show the probability
  density of music listening habits across age groups, and Step Charts to track discrete stock price movements.

### Day 8: Introduction to Pandas

- Introduced foundational `pandas` concepts for data manipulation and analysis.
- **Practical:** Constructed DataFrames from dictionaries and nested lists. Practiced utilizing essential exploration
  functions (`.info()`, `.describe()`, `.shape`) on student grade datasets, and executed column-wise mathematical
  operations to project future job salaries.

## Skills Acquired

* **Advanced Visualization:** Mastery of `matplotlib` functions including `fill_between`, `scatter`, `stackplot`,
  `stem`, `boxplot`, `step`, and `violinplot`.
* **Plot Customization:** Enhancing visual clarity using labels, legends, customized tick marks (`plt.xticks`), and
  transparency (`alpha`).
* **Pandas Data Structures:** Creating and structuring 1D Series and 2D DataFrames from various raw Python data types.
* **Data Exploration:** Inspecting dataset schemas, dimensions, and statistical summaries using built-in Pandas methods.
* **Data Manipulation:** Performing vectorized mathematical operations across DataFrame columns to derive new aggregated
  metrics.
