## Summary

Day 7 of the Data Analytics internship continued the exploration of data visualization using `matplotlib.pyplot`. The
session focused on advanced statistical charting techniques to visualize data distributions and discrete changes over
time, specifically covering box plots, violin plots, and step charts.

## Key Learnings

### Advanced Statistical Plots with Matplotlib

* **Box Plots:** Learned to utilize `plt.boxplot()` to visualize the distribution, central tendency, and potential
  outliers of multiple datasets side-by-side.
* **Violin Plots:** Explored `plt.violinplot()`, which combines the features of a box plot with a kernel density plot,
  providing a deeper understanding of the probability density of the data at different values.
* **Step Charts:** Implemented `plt.step()` to accurately represent variables that change at discrete intervals,
  utilizing parameters like `where='post'` to control step placement.
* **Axis Tick Customization:** Practiced replacing default numeric axis ticks with descriptive categorical labels using
  `plt.xticks()`, greatly improving chart readability.

## Practical Application

### Statistical Visualization Exercises

* **Departmental Salary Box Plot:** Created a box plot to compare the salary distributions across different corporate
  departments (IT, HR, and Sales), effectively highlighting variances and outliers.
* **Music Listening Habits Violin Plot:** Built a violin plot to analyze the distribution of time spent listening to
  music across distinct age groups (Teenagers, Young Adults, Middle Aged).
* **Stock Price Step Chart:** Developed a step chart to visualize the discrete daily closing prices of a stock over a
  work week, demonstrating how prices remain constant until the next interval.
